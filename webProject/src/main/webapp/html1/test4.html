이미지 삽입

웹에서 사용하는 이미지 형식
.jpg/.jpeg(사진 파일, 색상과 명암을 다양하게 표현)
.gif(색상수 최대 256가지를 이용, 크기가 작은 아이콘)
.png(다양한 색상을 표현, 투명 배경)


이미지 크기 단위
디스플레이 해상도 : PX(빛의 점 단위) 많을수록 해상도가 좋다.
px : 이미지 크기를 픽셀단위로 지정, 고정값
% : 이미지 크기를 % 지정, 웹 브라우저 너비, 높이 기준으로 이미지 크기를 표시.


이미지 경로가 같은 폴더에 없는 경우
폴더명/파일명 : 같은 폴더 내에 있는 이미지가 있는 경우
../폴더명/파일명 : 상위(부모) 폴더로 가기
../폴더명/폴더명/파일명 : 좀 더 상위 폴더에 이미지가 있는 경우
